# myleap

## A cta trading tool for crypto.

## 安装教程
### 软件安装 
1、安装 ta-lib
`conda install -c conda-forge ta-lib`

2、安装其他第三方包
`pip install -r requirements.txt`

### 启动软件
1、图形界面启动
`python run_ui.py`

2、命令行服务器启动
`python run_noui.py`

### 软件配置
重点关注用户目录下的两个文件夹：

配置文件夹（软件基本配置）：`.myleap`

策略文件夹(添加自己的策略)：`strategies`
